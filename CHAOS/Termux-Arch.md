# Termux 上通过 proot 安装 Arch
> 来自 SDRausty 的[**这个 repo**](https://github.com/SDRausty/TermuxArch)

## 安装 Termux
略

## 准备脚本
```
wget https://raw.githubusercontent.com/SDRausty/TermuxArch/master/setupTermuxArch.sh
chmod +x setupTermuxArch.sh
```

## 安装
```
./setupTermuxArch.sh
```
